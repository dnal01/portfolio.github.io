Google search Operators

  • site:siteName text text text
    • siteName will search for a site
    • siteName text text text will search for a specific text onto a specific site
  • "exactText will search only for this text combination
  • -text will exclude this word from a search

VSCode short keys

  • command + D Selects instances of a selected text one by one
  • command + shift + L Selects all the instances of a selected text
  • command + shift + return Creates a line above
  • shift + command + K Removes a line
  • command + [ / command + ] Indent/outdent line
  • command + / Toggle line comment
  • shift + option + mouse selection Column selection
  • command + page up / command + page down Scroll a page
  • control + G Got to Line...
  • command + N New File
  • command + P Got to File...

Intellij IDEA short keys

  • shift + Enter Start New Line
  • command + option + enterStart New Line Before Content
  • command + D Duplicate Line or Selection
  • option*2 + up / down Clone Caret Above / Below
  • option + shift + G Add Caret to Each Line in Selection
