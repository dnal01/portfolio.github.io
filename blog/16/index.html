Node.js / Express

Modules - introduction

Modules - libraries. In order to connect module store it in a variable var varName = require('moduleName');[1]

Modules - creation of your own module

In order to create your own module use exports keyword after which follows simple function syntax:

  • exports.myDateTime = function() {return...}; or module.exports = .... Store the code in a separate file and connect to it in the main file using:
  • var varName = require('./fileName without .js'). ./ - locate the module in the same folder as the main file. Then access the module in the main file:
  • varName.functionName()

Modules

  1. var varName = require('http'); transfer data over http
    2. Methods - http:
    • http.createServer(function(req, resp){..., incl. header}).listen(8080);. Creates server that listens to server ports and gives a response back to the client. The function passed into the http.createServer() method, will be executed when someone tries to access the computer on port ...
      • req argument - is an object (http.IncomingMessage object).
        • req properties
        • url holds the part of the url that comes after the domain name
        • ...
      • res argument -
      http Header

      If the response from the HTTP server is supposed to be displayed as HTML, you should include an HTTP header.

      • res.writeHead(200, {'Content-Type': 'text/html'});. The first argument is OK code, and the second is an object containing the response headers.
      • res.write('Hello World!'); or res.write(req.url);.
      • res.end(); appends whatever is in brackets to the end of the body
  2. var varName = require('url'); splits up a web address into readable parts
      3. url - 'http://localhost:8080/default.htm?year=2017&month=february'
    • var q = url.parse(req.url, true).query; - Use the url module to turn the querystring into an object
    • console.log(q.host); //returns 'localhost:8080'
    • console.log(q.pathname); //returns '/default.htm'
    • console.log(q.search); //returns '?year=2017&month=february'
    • var txt = q.year + " " + q.month; //"returns" '2017 february'
    • res.end(txt);
  3. var varName = require('fs'); allows you to work with the file system on your computer
    4. Methods - fs:
    • fs.readFile();
    • ...
