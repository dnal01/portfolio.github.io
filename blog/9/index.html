My takeaways today

Shell commands

  • date
  • echo hello print in shell one word
  • echo "Hello World" combination
  • echo Hello/ World
  • echo $PATH shows all libraries where it can find programs. And it is enough to just type a name of a program which is in those folders.
  • > rewrite output into something e.g. file. E.g. echo hello > hello.txt
  • >> append
  • which date shows in which directory this program "date" is located.
  • pwd print working directory
  • cd .. one directory up
  • cd ./foldername one directory down
  • cd - one directory down
  • \ previous directory
  • ls -l or -la or lh with "-" you can add arguments -la to show hidden and -h for size

  • if starts with d it is directory; -r for file. r = read. w = write. x = execute.

  • clear or ctrl+l - clean terminal
  • mv filename ../newFileName or mv filename foldername/filename destinFolderNamemove and rename
  • mkdir foldername create a directory
      • Flags
    • -v It displays a message for every directory created
    • -p It creates parent directories as necessary if they are missing, as using mkdir without -p and existence of those directories will cause an error message.
  • rm filename removes only files
  • rmdir emptyFolderName remove only empty folder
  • rm -r remove folder with files
  • program && another program execute more then one program with &&
  • cat filename or cat < filename prints into terminal all the content from a file
  • cp or fileName destFileName copies a file
  • cp -R orFolderName destFolderName copies a whole directory
  • touch filename creates a file like code command but without opening it
