My takeaways today

Overwriting default browser properties:

Set a margin and padding for a body(*) to 0 to overwrite default behavior of different browsers as they may be different and add no needed properties.

There is a border box and content box. If you want to set up width without influence of padding or borders use the below properties:

html {
    box-sizing: border-box; // another option content-box
}
*, *:before, *:after {
    box-sizing: inherit; // to inherit border-box property for child elements of html
}

Regular expressions

To validate or extract information from input

  • . any single character (except line terminators)
  • * zero or more times
  • + one or more times of a next after that character
  • ? 0 or 1 time
  • {n}n occurrences
  • {n,m} at least n occurrences, at most m occurrences
  • [0123456789]
  • [0-9] any one of the range of characters
  • \d any digit
  • \D any character that is not a digit

Example:

^[a-zA-Z0-9.!#$%&'*+\/=?^_`{|}~-]+@[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}[a-zA-Z0-9])?(?:\.[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}[a-zA-Z0-9])?)*$

CSS - Flex-box
div.parent-flex2 {
    display: flex;

    flex-direction + flex-wrap > flex-flow
    flex-flow: row nowrap; // OR:
    flex-flow: column wrap;
    flex-flow: column wrap-reverse;

    justify-content: space-around; // Horizontally OR:
    justify-content: flex-start: Items align to the left side of the container.
    justify-content: flex-end: Items align to the right side of the container.
    justify-content: center: Items align at the center of the container.
    justify-content: space-between: Items display with equal spacing between them.
    justify-content: space-around: Items display with equal spacing around them

    align-items: flex-start: Items align to the top of the container. Vertically OR:
    align-items: flex-end: Items align to the bottom of the container.
    align-items: center: Items align at the vertical center of the container.
    align-items: baseline: Items display at the baseline of the container(content).
    align-items: stretch: Items are stretched to fit the container.

    align-self: the same parameters and is for the specific item.

    flex-direction: row: Items are placed the same as the text direction. OR:
    flex-direction: row-reverse: Items are placed opposite to the text direction.
    flex-direction: column: Items are placed top to bottom.
    flex-direction: column-reverse: Items are placed bottom to top.

    order: -2; OR -1, 0, 1, 2.

    flex-basis: 10%: but a little more or less than that is OK.
    
    flex-grow and flex-shrink;

    align-content;
}
