My takeaways today

During the time from previous post I have learned:

  1. Animations. @keyframes tag in css is responsible for animation. You also need to name the animation, simply adding the name before opening brackets. Then make some configuration and in the css class add animation-name: and other animation attributes.
  2. Flex. With flex you can position content as you want: vertically and horizontally: add spaces between; create cards or represent boxes in reverse order using justify-content property. There is a good article I have placed below. I have tried that with the blog section, check it out below.
  3. Box-sizing. But not that the tag itself. Using display:flex in content (child element) itself can realize entire parent container to the size of child element, if parent element did no have sizes specified. Thats cool because I do not need to specify sizes twice.
  4. href. IDs are useful for inserting a link which directs directly to them on a page. For that when setting up a link, after .html without spaces add # and id name. Or if referencing to the current page then just # and id name.
  5. overflow-x: scroll or auto. overflow with display: flex creates a block where you will need to scroll.
  6. Responsive website. @media all and (min-width: 1000px) {class {parameters which will be replaced if width is more then 1000px}}.
  7. Additional Tags:
    • aside - for information that does not really belong to the page.
    • span - for adding extra css code.
    • pre - for bunch of code. Format stays as it was written.
  8. Page layouts. display: grid - Container layout. First add it to a container together with grid-template-rows:; grid-template-columns:; grid-template-areas:;. And then add grid-area:; into a child element: 
    
/* Parent element */
.myPageLayoutGrid {
display: grid;

/* Additional configurations */
grid-column-gap: 0.5em;
grid-row-gap: 1em;
/* Additional configurations */

grid-template-rows: auto;
grid-template-columns: 1fr 2fr 7fr;
grid-template-areas:
    "egArticle egAside2 egAside3"
    "egArticle egAside2 egAside3"
    "egAside1 egAside1 egAside1";
}

/* Child elements */
.myGridArticle {
    grid-area: egArticle;
}...
  9. New CSS styles:
    • text-wrap: - wraps text. Useful for pre tag, when, by default, warping is disabled.

      • Special effects:

    • transform: translateY() moves a block.
    • rgba where a - is a transparency value between 0 and 1, where 0 is completely invisible
    • transition: transition-property, transition-duration, transition-timing-function, transition-delay, and transition-behavior. It is better to add to a class itself to reverse changes smoothly.
    • :active like :hover but adds effect when click.
    • :target. Gets applied whenever the target of the last link clicked.
    • position: fixed. The position will stay put when you scroll.
    • visibility: hidden. Hide an element.
  10. Photo collage. Make a container for photos and then position each photo where would you like by using different classes. Use z-index to set some on front, the higher the closer to the front.
  11. Lightbox. It is composed of two links. In the first link you are going to include what you want to appear after you click on a second link: picture, some text, etc. The first link should has href="#_" and an id. The second link - consist of what you will click on to activate a first link. The second link should have href="#id of a first link". Then in CSS there are three major things to be included in class of a lightbox: position: fixed; visibility: hidden: z-index: 999; then add :target and add visibility: visible
