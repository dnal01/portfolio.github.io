JSON

JSON - text file. Used to communicate between server and browser. Can not store function, date and undefined. But can store functions as a string. Convert the function into a string (e.g. "age" stores function):

obj.age = obj.age.toString();

And later convert them using in addition eval, but avoid using it

obj.age = eval("(" + obj.age + ")");

JSON Syntaxis

'{"name":"John", "age":30, "car":null}' - Object

'["name", "age", "car"]' - Array

JSON methods

JSON.stringify(jsData); - to convert it into a string

JSON.parse(jsonData) - to convert text into a JavaScript object
