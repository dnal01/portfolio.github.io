getElement(-s)By...
  • document.getElementById("id")
  • document.getElementsByClassName("classname")
  • document.getElementsByTagName("tagname")

    • Both of the last two return all of the elements as a list. Thus, you can call single elements by their index, e.g. [0] or loop through them.

    Example1: const x = document.getElementById("main");

    Example2: const y = x.getElementsByTagName("p");

    This example finds the element with id="main", and then finds all "p" elements inside "main"
