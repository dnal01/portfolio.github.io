    • Working with lists

    • varName[-1] access a list from the end
    • varName[3:5] access a list in diapasone by the rule [ start - inclusive : end - exclusive]
    • varName[:5] access a list in diapasone starting from index 0

      • Cooping lists

    • newVar = list(existingVar) or newVar = existingVar[:] creates a new list in memory, becouse if using "=", the new variabe will point to the same adress in memory where the existed list was.

      Note any other operators, where you do not simply asign variables, but performe some operations, will also create a new list, e.g. newVar = existingVar + [newEl1, newEl2]
